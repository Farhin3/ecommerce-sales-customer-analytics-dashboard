# Exploratory Data Analysis (EDA) Report

## Project Objective

Analyze e-commerce transactions to evaluate sales performance, profitability, customer behavior, and regional trends.

## Dataset Overview

The dataset contains e-commerce transactions with:

- Order Information
- Customer Information
- Product Categories
- Sales and Profit Metrics
- Regional Details
- Payment Modes

## Data Cleaning Performed

- Checked missing values
- Checked duplicate records
- Verified data types
- Validated sales and profit fields

## Exploratory Data Analysis

### Sales Analysis

- Total Sales
- Sales by Category
- Sales by Region
- Monthly Sales Trend

### Profitability Analysis

- Total Profit
- Profit by Category
- Profit Margin Analysis
- Low Profit Categories

### Customer Analytics

- Top Customers
- High-Value Customers
- Customer Purchase Patterns

### Regional Performance

- Region-wise Sales
- Region-wise Profit
- City Performance Analysis

### Payment Analysis

- Payment Method Distribution
- Most Used Payment Modes
